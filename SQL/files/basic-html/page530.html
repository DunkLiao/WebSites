This appen%u2010dix is meant to serve as a brief overview of how window functions work. Windowfunctions make many typically difficult tasks (difficult to solve using standard SQL,that is) quite easy. When you useGROUP BY in a query, each row in the result set is a group and represents one ormore rows with the same values in one or more columns that you specify.
