一、概述

2025年6月，台灣金融生態系面臨多重嚴峻的資安挑戰，從國家級駭客組織的滲透、供應鏈勒索、高階社交工程，到廣泛的偽冒App/網站與釣魚簡訊詐騙，顯示威脅已演變為「供應鏈滲透」、「高階社交工程」與「資料脅迫勒索」為主流。

關鍵威脅趨勢

面對日趨複雜的攻擊模式，金管會、FISAC與金融業者正加速推動「AI防詐」與「零信任」等強化措施，並取得關鍵進展，以提升整體防禦韌性。

二、重大事件回顧與威脅態勢分析

本月發生多起具指標性的資安事件，反映出當前的主要威脅趨勢：

國家級駭客攻擊加密貨幣平台

幣託（BitoPro）事件

  • 損失金額：約1,150萬美元加密貨幣
  • 攻擊者：北韓Lazarus集團
  • 攻擊手法：社交工程→EDR旁路→雲端令牌竊取→橫向移動
  • 關鍵突破：劫持AWS Session Token，繞過MFA

影響分析：突顯「繞過MFA＋雲端權限升級」的新型鏈上盜竊手法，以及雲端權限管理的脆弱性。

勒索軟體鎖定保險與上市櫃公司

Kiwi86保險系統攻擊

  • 攻擊者：DireWolf勒索團體
  • 受影響資料：152萬名保戶與618萬筆保單
  • 威脅策略：分階段公開資料施壓
  • 攻擊模式：雙重勒索策略

其他重大案例：

  • 黑松（1234）：內部郵件通訊錄外流，418筆帳密暗網釋出
  • 熒茂光學（4729）：遭加密勒索攻擊

網路釣魚與高階社交工程升級

即時網釣技術（AiTM攻擊）

從傳統非即時網釣轉向「即時網釣技術」，能夠同步竊取OTP驗證碼，大幅提升攻擊成功率。

主要詐騙手法分布：

  • 水費欠款通知：21%
  • 電信點數詐騙：19%
  • 交友軟體詐騙：13%
  • 投資詐騙：其他

高風險族群：41至60歲年齡層的中年女性成為主要受害群體

國際資安事件警示

伊朗銀行遭駭事件

伊朗賽帕銀行遭「國家級網路作戰行動」毀滅性攻擊，導致系統癱瘓、線上服務與ATM中斷、用戶存款歸零，全國金融體系崩潰。

史上最大規模DDoS：

Cloudflare成功阻擋高達7.3 Tbps規模的DDoS攻擊，目標為主機代管業者，創下歷史新高。

三、防禦與監理動態

面對日益嚴峻的威脅，台灣金融業正積極部署多層次防禦措施：

AI驅動的防詐實戰部署

  • ATM防詐：兆豐銀率先導入「臉部遮蔽辨識＋通話行為偵測」
  • 簡訊防詐：公股八行第三季全面啟用「680xx」金融簡訊專碼
  • 客戶服務：北富銀與富邦金等61家機構導入LLM與AI模型
  • 跨行協作：金融科技產業聯盟啟動「AI金流履歷」PoC

預期效益

預計2026年前建立跨行詐欺早期預警共享模型，大幅提升防詐效能。

零信任架構落地

  • 監理指引：金管會「金融業導入零信任架構參考指引」正式生效
  • 成熟度模型：結合FCB/GCB基準，提供四階段成熟度模型
  • 推動機制：臺灣證券交易所舉辦說明會推動證券商導入

核心原則

  • FIDO身分鑑別與身分認證
  • 設備合規管理
  • 設備安全判斷與應處機制

供應鏈與雲端資安強化

  • 外部攻擊面評估：建議導入Bitsight等工具，量化第三方風險
  • 雲端安全：強化雲端環境的資安監控與管理
  • DevOps安全：針對DevOps環境的安全配置加強監控

多雲趨勢

Fortinet報告顯示逾八成企業採用多雲或混合雲，金融業應強化相應的資安防護措施。

情資分享與聯防機制

  • FISAC成效：會員數達332家，聯防SOC涵蓋64家機構
  • 情資分享：會員自供比例已突破50%
  • 預警機制：針對偽冒銀行假網址連續預警

揭露趨勢

上市櫃重大訊息統計顯示2024全年資安事件已超過前兩年總和，2025首季再成長兩倍，反映揭露意識與攻擊次數同步攀升。

四、對金融機構之建議

綜合上述分析，為應對當前資安威脅，建議金融機構採取以下措施：

雲端權限管理

  • 禁用長時效AWS Token
  • 導入Just-in-Time MFA
  • 啟用CloudTrail與GuardDuty行為基準
  • 強化雲端行為監控

供應鏈與外包治理

  • 保險與證券平台須落實ISO 27001與SOC 2 Type 2規範
  • 導入Bitsight等外部攻擊面評分工具
  • 量化第三方風險
  • 強化對第三方供應商的資安要求與監督

零信任落地

  • 以「雲端存取、系統維運、遠距辦公」為首波場域
  • 加速導入零信任架構
  • 依金管會分級指標逐季檢核
  • 確保零信任防護實施到位

AI防詐與認證強化

  • 跨行共享黑名單與詐欺特徵
  • 整合680xx短碼、Whoscall API
  • 導入ATM人臉遮蔽辨識與交易行為異常偵測
  • 採用FIDO2標準、生物辨識等安全認證方式

資安事件揭露與應變

  • 遵循證交所最新FAQ
  • 於30分鐘內發布「撞擊營運或疑似資料外洩」重訊
  • 建立備援通報流程
  • 避免勒索導致遮蔽揭露

提高資安意識培訓

  • 針對中年女性等高風險族群加強宣導
  • 提升對網路釣魚等社交工程攻擊的警覺性
  • 定期舉辦資安教育訓練
  • 建立資安意識評估機制

優先行動項目

建議金融機構優先關注雲端權限管理與零信任架構導入，這兩項措施能夠有效應對當前最主要的威脅向量。

五、結論

威脅態勢總結

2025年6月的連環攻擊證明，「國家級駭客、勒索團體與詐欺產業鏈高度交織，傳統邊界防禦難以應付。」「雲端權限、第三方平台與高階社交工程成為金融業最大風險。」

關鍵洞察

「AI與零信任不再只是技術名詞，而是既定監理要求與實務剛需。」面向下半年，金融機構應在「雲-端-終端」強化身份驗證與行為分析，同時加深業界情資分享與跨域聯防，方能在威脅規模指數化成長下維持營運韌性與民眾信任。

未來展望

  • 持續投資資安防護技術
  • 採用新興技術如AI輔助資安分析
  • 加強跨產業資安情資共享
  • 提升整體防禦韌性

行動呼籲

  • 立即檢視雲端權限管理機制
  • 加速零信任架構導入時程
  • 強化員工資安意識培訓
  • 建立完善的事件應變流程
